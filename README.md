# Rose Day — small responsive site

Open `index.html` in a browser. It's a small 3-page responsive site with a playful escaping "No" button and a celebratory message + confetti when clicking "Yes".

Quick steps (Windows):

1. Open the folder in Explorer: `c:\Users\Lenovo\Desktop\secret`
2. Double-click `index.html` to open in your browser, or serve with a simple static server.

Optional (serve with Python):

```bash
python -m http.server 8000
# then open http://localhost:8000/index.html
```

Files:
- `index.html` — main interactive page
- `memories.html`, `contact.html` — small extra pages
- `styles.css`, `script.js` — styling and behavior
